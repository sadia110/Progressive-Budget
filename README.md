
# Employee-Tracker   

## Description 
Buget App that allows user to enter their data while being online and offline. 

## Tech 
Mongoose Db  

## Usage 
Fork respo, in command line install :
- npm install express 
- npm install mongoose  
Then run:  node server.js    

runs on localhost:3000
